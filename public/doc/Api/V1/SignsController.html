Signs [1.0]

Managing access token

create

Get access token depend on user credentials

URL

POST api/v1/sign

Required Parameters

  • email String

    Email of given user
  • password String

    Password of given user

Examples

Request

Headers

  • Content-Type:application/json
  • Accept:application/json
 {
  "data": {
    "type": "users",
    "attributes": { 
      "email": "user@test.com",
      "password": "password"
    }
  }
 }
Response 
 {
   "data": {
       "id": "4",
       "type": "access-token",
       "attributes": {
           "access-token": "e9ccc461a53f"
       }
   }
}
