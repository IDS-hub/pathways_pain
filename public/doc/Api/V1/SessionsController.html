index

Show list of available sessions * _feel_good_ - where session history of given pain cause between 0 and 1

URL

GET /api/v1/pain_causes/:pain_cause_id/sessions

Required Parameters

  • pain_cause_id Integer

    Id of pain cause

Optional Parameters

  • session_type String

    Type of session. Avalilable:

Examples

Request

GET /api/v1/pain_causes/12/sessions?session_type=feel_good

Headers

  • Content-Type:application/json
  • Accept:application/json
  • Access-Token:accesstoken
