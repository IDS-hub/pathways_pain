SessionHistories [1.0]

User session history

index

Show current user session history

URL

GET api/v1/profile/session_histories

Optional Parameters

  • session_id Integer

    Id of selected session
  • pain_cause_id Integer

    Id of pain cause
  • sort_order String

    Sort order. Allowed: 'asc', 'desc'. Default is desc
  • is_chart Any

    Show short list with minimun fields if param is present

Examples

Request with all optional params filled for chart Request

GET /api/v1/profile/session_histories?session_id=1&pain_cause_id=37&is_chart=true

  • Content-Type:application/json
  • Accept:application/json
  • Access-Token:accesstoken
