create

Sign up new user

URL

POST api/v1/profile

Required Parameters

  • email String

    User email
  • password String

    User password
  • password_confirmation String

    Same value as password field

Optional Parameters

  • first_name String

    User first name
  • last_name String

    User last name
  • is_skip_verify Boolean

    (default: true). Skip sending verification email to entered email if true

Examples

Request

Headers

  • Content-Type:application/json
  • Accept:application/json
{
  "data": {
    "type": "users",
    "attributes": { 
      "email": "vvvvvv_test@test.com",
      "first_name": "first",
      "last_name": "last",
      "password": "password",
      "password_confirmation": "password",
      "is_skip_verify": false
    }
  }
}
Response Null if not skipping verification. Otherwise will be new user data
