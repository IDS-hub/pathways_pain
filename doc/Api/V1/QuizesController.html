Quizez

index

Show list of available quizez for given session

URL

GET api/v1/pain_causes/:pain_cause_id/sessions/:session_id/quizes

Required Parameters

  • pain_cause_id Integer

    Id of given pain cause
  • session_id Integer

    Id of given session

Examples

Request

GET api/v1/pain_causes/1/sessions/2/quizes

Headers:

  • Content-Type:application/json
  • Accept:application/json
  • Access-Token:accesstoken
 Response 
{
  "data": [
      {
          "id": "2",
          "type": "quiz",
          "attributes": {
              "question-text": "Is it possible to cure pain cause without doctors?",
              "answer-text": "Yes, but it is not healthy",

              "position": 1
